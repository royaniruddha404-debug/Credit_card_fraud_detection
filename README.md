Credit Card Fraud Detection

Problem Statement
Detect fraudulent credit card transactions from a highly imbalanced dataset.

Tools and Technologies
Python
Pandas, NumPy
Scikit-learn
Imbalanced-learn (SMOTE)
XGBoost, Logistic Regression

Methodology
Performed data preprocessing and feature scaling
Handled severe class imbalance using SMOTE
Trained Logistic Regression and XGBoost models
Evaluated models using ROC-AUC, Precision-Recall curve, and confusion matrix
Tuned decision thresholds to improve fraud recall

Results
Achieved ROC-AUC of 0.97 using XGBoost

Business Impact
This model helps financial institutions detect fraudulent transactions early, reducing financial losses and improving customer trust.
